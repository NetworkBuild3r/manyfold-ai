# Security Policy

## Supported Versions

At least until we reach v1.0, only the latest release version is supported with security updates.

We apply security patches when available; track `latest` and check for updates regularly.

## Reporting a Vulnerability

If you find a security vulnerability in this project, please report it confidentially by opening a draft security advisory in this repository (GitHub: **Security** tab → **Advisories** → **New draft**), or contact the maintainers through the channel listed in the repo (e.g. an issue with a request for private disclosure).

We will publish and credit security advisories and fixes when it is safe to do so.
